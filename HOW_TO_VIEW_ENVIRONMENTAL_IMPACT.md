# 📍 "나의 환경 기여도" 보는 방법

## 🎯 표시 위치

**화면**: 지도 탭 → 녹지시설 마커 클릭 → LocationModal

**구체적인 경로:**
1. 하단 네비게이션에서 **"숲세권 지도"** 탭 클릭
2. 지도 위의 **녹지시설 마커** 클릭 (공원/산책로/쉼터)
3. 모달이 열리면 아래로 스크롤
4. **"나의 환경 기여도"** 섹션 확인

---

## 📋 표시되는 내용

### 1. 오늘 절감한 CO2
- 습관 완료에 따른 CO2 절감량 (kg)
- 예: "1.05kg"
- 메시지: "대단해요! 3개 습관으로 1.05kg CO2 절감! 나무 19.1그루가 하루 동안 흡수하는 양이에요."

### 2. 나무 흡수량 환산
- CO2 절감량을 나무 심기 개수로 환산
- 예: "19.1그루"
- 설명: "나무 흡수량 (그루/일)"

### 3. 자동차 주행 절약
- CO2 절감량을 자동차 주행 거리로 환산
- 예: "8.8km"
- 설명: "자동차 주행 절약"

### 4. 장소 방문 효과
- 해당 장소 방문 시 추가 환경 효과
- 예: "이 장소에서 2.4g 추가 효과!"
- 장소별 맞춤 메시지

---

## ⚠️ 중요: 습관을 먼저 완료해야 표시됨!

**"나의 환경 기여도" 섹션은 습관을 완료해야만 표시됩니다.**

### 보는 방법 (단계별):

1. **습관 트래커 탭으로 이동**
   - 하단 네비게이션에서 **"에코 습관"** 탭 클릭

2. **습관 체크하기**
   - 체크리스트에서 습관을 체크
   - 예: "텀블러 사용하기", "대중교통 이용하기" 등

3. **지도 탭으로 돌아가기**
   - 하단 네비게이션에서 **"숲세권 지도"** 탭 클릭

4. **녹지시설 마커 클릭**
   - 지도 위의 공원/산책로/쉼터 마커 클릭

5. **모달에서 확인**
   - 모달이 열리면 아래로 스크롤
   - **"나의 환경 기여도"** 섹션 확인
   - 습관을 완료했다면 섹션이 표시됨!

---

## 🔍 섹션이 안 보이는 경우

### 1. 습관을 완료하지 않았을 때
- **증상**: "나의 환경 기여도" 섹션이 아예 안 보임
- **해결**: 습관 트래커에서 습관을 체크한 후 다시 확인

### 2. 습관을 완료했는데도 안 보일 때
- **확인사항**:
  - 습관 트래커에서 습관이 체크되어 있는지 확인
  - 지도 탭으로 이동한 후 마커를 다시 클릭
  - 모달을 아래로 스크롤해서 확인

### 3. 데이터가 저장되지 않았을 때
- **증상**: 습관을 체크했는데 반영이 안 됨
- **해결**: 
  - 브라우저 개발자 도구에서 localStorage 확인
  - 시크릿 모드가 아닌지 확인
  - 페이지를 새로고침

## 💡 팁

1. **습관을 많이 완료할수록 더 많은 CO2 절감**
   - 1개 습관: 약 0.1-0.8kg
   - 3개 습관: 약 1.0kg
   - 8개 습관 (모두 완료): 약 3.0kg 이상

2. **장소별로 다른 효과**
   - 탄소 흡수량이 높은 장소: 더 많은 추가 효과
   - 열쾌적성이 높은 장소: 에너지 절약 효과

3. **실시간 업데이트**
   - 습관을 체크하면 즉시 반영됨
   - 지도 탭으로 돌아가서 마커를 다시 클릭하면 업데이트된 값 확인 가능
